获取你的 AWS 环境

在 AWS 活动中运行本 Workshop

Workshop Studio 会为你提供一个临时 AWS 账号来完成本训练营。这个账号有一些使用限制,但完成实验后你可以自由探索和实验。账号会在活动结束时自动清理,不会对你的个人账号产生任何费用。

按以下步骤获取你的环境。

步骤

  1. 打开 Workshop Studio  或使用讲师提供的链接

  2. 点击 Email one-time password

  1. 输入你的邮箱地址,点击 Send passcode

系统会发一封包含一次性密码的邮件到你的邮箱。复制邮件中的验证码用于下一步。

没收到邮件?

请检查垃圾邮件文件夹。如果仍然没有,请联系现场讲师确认你的邮箱已加入白名单。

  1. 输入邮件中收到的验证码,点击 Sign in

  1. 输入讲师提供的 Event access code,点击 Next。如果你使用的链接已包含 event code,可以跳过这一步。

  1. 阅读 Terms and Conditions,勾选 I agree with the Terms and Conditions,点击 Join event

进入 AWS 控制台

加入 event 后,你会看到 workshop 的步骤列表在左侧。

点击左下角的 Open AWS console 进入你的临时 AWS 账号。建议复制这个登录 URL 并在新窗口中打开。

不要自己在浏览器输入 console.aws.amazon.com——那样会要求输帐号密码。始终通过左侧 Open AWS console 链接进入。

你现在应该已经可以访问临时 AWS 账号了。注意这是临时账号,活动结束后会自动清理。Workshop Studio 顶部会显示剩余时间。

环境就绪,进入下一步开始动手。
