# 可选延伸 (Optional)

**形式**: 可选动手 | **时长**: 各约 10-15 分钟 | **是否必做**: 否

---

## 这是什么

你已经走完了核心的六个 Phase——从定义"好"、构建、评估、诊断、优化到治理,完整转了一圈 ADLC 飞轮。**到这里主线就完整了**,时间有限可以直接跳到下一页"总结"。

下面两个是**可选延伸**,给学有余力、想再深入一点的体验者。它们不引入新概念,而是把已经学到的评估能力用到两个很 CXO 的问题上:

| 延伸 | 回答的问题 | 时长 |
| --- | --- | --- |
| A. 多模型对比 | "换个更便宜的模型,质量还过得去吗?" | ~10-15 分钟 |
| B. 评委稳定性 | "你这个 AI 裁判自己靠谱吗?会不会瞎打分?" | ~5-10 分钟 |

!!! info
    **前提**:这两个延伸都要用到你前面部署好的 Agent 和评估器,所以**请在跑"清理资源"之前做**。跑完清理后资源就没了,这两个就跑不了了。

!!! warning
    这两个延伸不在 2 小时主线预算内。如果是带教场次、时间紧,讲师可以只演示、或让学员课后自行体验。

挑你感兴趣的开始,或者直接跳到总结。
